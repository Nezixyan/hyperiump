# Commands to download panel

Update repositories
```html
apt update
```

Install GIT
```html
apt install git
```
Install CURL
```
apt install curl
```

Install NodeJS
```
curl -sL https://deb.nodesource.com/setup_16.x | sudo bash -
sudo apt -y install nodejs
```
Install panel
```html
git clone https://github.com/Nezixyan/hyperiump.git
```

```
cd kubek-minecraft-dashboard
```

```
npm install
```

```
npm start
```
